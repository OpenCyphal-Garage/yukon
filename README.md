Yukon is a rich integrated development environment (IDE) for developing and maintaining
[Cyphal](https://opencyphal.org)-powered vehicular computing systems.
It is designed to run on GNU/Linux, Windows, and macOS.

**Download pre-built executables from [GitHub Releases](https://github.com/OpenCyphal/yukon/releases).**

⚠ While Yukon is usable, it is in the **alpha stage** and is known to contain bugs.
Please use responsibly and report all issues you encounter via the [bug tracker](https://github.com/OpenCyphal/yukon/issues).

## Minimal system requirements

* Screen resolution 1920x1080
* RAM 8 GiB
* 4x CPU (cores)
* SSD
